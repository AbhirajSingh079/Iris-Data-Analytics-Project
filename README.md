# Iris Data Analytics Project

##  Objective
Analyze the Iris dataset and build a classification model.

##  Steps
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Logistic Regression Model

## Insights
- Petal features are highly important
- Setosa is easily distinguishable
- Model achieved high accuracy

## Tools Used
Python, Pandas, Seaborn, Scikit-learn
